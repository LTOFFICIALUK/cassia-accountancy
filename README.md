# Cassia Accountancy Website

A modern, professional website for Cassia Accountancy — helping small business owners with bookkeeping, tax returns, cloud accounting and Making Tax Digital support.

## Tech Stack

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

| Page | Route |
|------|-------|
| Home | `/` |
| About | `/about` |
| Services | `/services` |
| Making Tax Digital | `/making-tax-digital` |
| Get a Quote | `/pricing` |
| Resources / Blog | `/blog` |
| Contact | `/contact` |
